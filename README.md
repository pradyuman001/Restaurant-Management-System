# Restaurant Management System

Restaurant Management System Application is provides you Live news using Free News API. I had use Google news API and Restful API. I used Retrofit method call API. Live News Application was made using Kotlin Lanugage.

The codebase focuses 👓 on following key things:

Code structuring, style and comments
Dependency injection
Offline first ✈️
Kotlin
And tests 🛠
Emojis (ofcourse) 😛

In Live News Application you can read news in very differnt Category. I used Query API to call same API with differnt parameters. In Live News Application news are refreshs in every 10 min.
